Formatação de texto

Principais formatações

Negrito / destaque

Para usar negrito uso a tag b (não semantica )

usando a tag strong tenho o mesmo efeito (tag é semantica/ tem significado)

para o termo ficar em italico uso a tag i (NÃO SEMANTICA/ forma)

usando a tag EM tenho o msm efeito, porem com semantica

Marcar texto

para deixar o texto marcado uso a tag mark

para deixar o texto grande uso a tag big(obsoleto) e para deixar pequeno uso a tag small

para deixar o texto sublinhado uso a tag u (n semantica) ou ins (semantica)

para da efeito de queda ex 2 em 52=25 uso a tag de texto sobrescrito ou efeito de queda como o 2 em H2O
